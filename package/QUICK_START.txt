=====================================
  INVENTORY TOOLS - QUICK START
=====================================

HOW TO USE:

1. Double-click "Inventory Tools" icon on your desktop
2. Wait 30-60 seconds for the emulator to start
3. Use the web interface that opens in your browser

That's it!

-------------------------------------
TROUBLESHOOTING:
-------------------------------------

"Emulator not found" error?
  -> Ask IT to set up Android Studio

"Device not connected" error?
  -> Wait a bit longer, emulator is still booting

Browser didn't open?
  -> Go to: http://localhost:5000

Need help?
  -> Contact IT support
